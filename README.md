## 🚀 Notion Blog — DevSecOps Project

Blog technique personnel connecté à Notion, construit avec une approche **DevSecOps** complète.

### Stack

- **App** : Next.js + Notion API
- **Conteneurisation** : Docker (multi-stage, Distroless)
- **CI/CD** : GitHub Actions + Trivy (scan sécurité)
- **Infra** : Terraform (AWS) + Ansible
- **Registry** : GitHub Container Registry (GHCR)

### Objectifs

- Apprendre le DevSecOps de bout en bout
- Mettre en pratique Git, Docker, CI/CD et Ansible
- Déployer une application en production sur AWS

> Projet d'apprentissage réalisé en mode Agile — chaque fonctionnalité est tracée via des User Stories et des Pull Requests.
